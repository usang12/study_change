시맨틱 마크업이란 HTML 태그로 문서를 작성하는 것을 말한다. 즉 의미를
잘 전달하도록 문서를 장성 하는 것이다. 특징으로 검색엔진이 시맨틱
태그를 중요한 키워드로 간주를 하기 때문에 검색엔진 최적화에 유리하다
또한 웹 접근성 측면에서 의미를 잘 파악할 수 있다. 마지막으로 코드를
볼 때 가독성이 더 좋다. 마지막으로 예를 들어 각 태그를 그 용도에 잘
사용해야 하는데 헤더/푸터에 <header>/<footer> 사용 하는 식으로 가진
의미에 맞게 사용하는 것 들을 말한다.

CSS Box Model은 HTML element가 웹 페이지에서 차지하는 공간을
정의한 모델이다. HTML element는 각각의 영역을 갖고 있으며 그 영역은
다시 여러개의 작은 영억으로 나뉜다 구선은 각 element는 가운데 실제 
내용이 담긴 content 감싸고 있는 border 그 사이의 영역인 padding
boder 바깥의 영역인 margin으로 구성되어 있다 각 영역들은 CSS의
property로 구현되어 있고 따라서 원하는 속성을 불러와 원하는 값을
대입하여 주면 된다. 크롬에서 F12를 눌러 이러한 구현 내용들을 확인
가능하고 그림으로 까지 표현해주는 것을 볼 수 있으므로 좋은 예시이다.

CSS flexbox는 felxbox의 인터페이스 아이템 간 공간 배분과 정렬 기능을
제공하기 위한 1차원 레이아웃 모델로 설계되어있다. 여기서 왜 1차원
이라고 칭 했냐면 레이아웃을 다룰 때 한번에 하나의 행 or 열을 다루기
때문입니다. 그리고 이 flebox를 잘 다루기 위해서는 주축과 교차축이라는
두가지의 축에 대한 정의를 잘 알아야 한다. 주축은 flex-direction 속성을
사용해 지정하며 교차축은 이에 수직인 축으로 결정된다. flexbox의 동작은
결국 이 두개의 축에 대한 문제로 환원 되기 때문에 어떻게 동작하는지
처음부터 잘 이해하고 사용하는 것이 중요하다. 다음으로 주축에 대한 설명을
이어가자면 주축은 flex-direction에 의해 정의되고, row, row-reverse, 
column, column-reverse라는 4개의 값을 가질 수 있습니다.
